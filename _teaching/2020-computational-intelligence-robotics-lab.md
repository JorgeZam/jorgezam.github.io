A two-part undergraduate laboratory course. As teaching assistant, I contributed 
to the computational intelligence section, covering pattern recognition techniques 
and their practical applications. Topics included classical pattern detection 
algorithms, feature extraction, and deep learning-based object detection using 
neural networks such as YOLO. Taught from 2020 to 2025.

## Technologies and Tools
- **Language:** Python
- **Frameworks:** TensorFlow / PyTorch
- **Object detection:** YOLO
